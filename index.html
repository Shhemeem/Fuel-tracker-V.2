Connection Issue Detected

The app cannot save data. This usually means the Firestore Database has not been created in your Firebase Console yet.

How to Fix (Required):

  1. Go to the Firebase Console.
  2. Select your project (milege-track).
  3. In the left menu, click Build -> Firestore Database.
  4. Click the Create Database button.
  5. Select a location (e.g., nam5 (us-central)) and click Next.
  6. IMPORTANT: Choose Start in test mode (allows read/write).
  7. Click Create.
